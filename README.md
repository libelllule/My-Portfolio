# Портфолио: Анна Разинкина 

Привет! В этом репозитории собраны ссылки на мои проекты. Они демонстрируют мои навыки в системном анализе, управлении продуктом и проектировании архитектуры. 


### 1. [Quick Share (My_qsh)](https://github.com/libelllule/My_qsh) — Локальный файлообменник
В этом командном проекте я выступила в роли **Product Manager и Системного аналитика**. Я разработала проектную документацию (BRD, SRS), спроектировала диаграммы последовательностей (UML Sequence), а также управляла бэклогом, ставя задачи разработчикам.**Мои материалы и схемы находятся в папке `docs` этого репозитория.**

### 2. [Кейсы: Бизнес- и Системный анализ](https://github.com/libelllule/Technical-Specification-BA-SA-Analyst)
Включает BPMN-модель закупки ИТ-оборудования с проведением Gap-анализа требований, а также UML-диаграмму (Activity) для алгоритма публикации товара на маркетплейсе.

### 3. [Система бронирования отелей](https://github.com/libelllule/BookingHotelSystem) 
Разработка бэкенда (Java + PostgreSQL). Проект показывает навык проектирования реляционных баз данных и написания SQL-запросов для обработки конфликтов при пересечении дат бронирования.

### 4. [AuthSystem](https://github.com/libelllule/AuthSystem) — Демон аутентификации (C/Linux)
Клиент-серверное приложение на C (интернет-сокеты TCP/IP + UNIX-сокеты). В проекте реализован надежный обмен данными по сети и обеспечена безопасность соединения (настроен HTTPS через OpenSSL).
